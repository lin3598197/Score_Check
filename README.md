# 期末成績計算機 📊

一個幫助學生計算期末考需要考幾分才能及格或免補考的互動式工具。

## 功能特色

- 🎯 **即時計算** - 輸入段考和平時成績，立即得知期末考需求
- 📈 **視覺化進度** - 清楚顯示目前累積積分
- ⚖️ **彈性計算** - 平時成績未知時，可計算合計需求
- 🎨 **直覺介面** - 簡潔美觀的使用者介面

## 成績權重

| 項目 | 權重 |
|------|------|
| 第一次段考 | 20% |
| 第二次段考 | 20% |
| 平時成績 | 30% |
| 期末考 | 30% |

## 使用方式

1. 輸入第一次段考分數 (0-100)
2. 輸入第二次段考分數 (0-100)
3. 選填平時成績 (若不知道可留空)
4. 系統會自動計算達到 60 分 (及格) 和 40 分 (補考門檻) 所需的期末考分數

## 技術架構

- **前端框架**: React
- **樣式**: Tailwind CSS
- **圖示**: Lucide React

## 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev
```
